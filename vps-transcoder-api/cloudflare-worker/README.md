# YOYO Streaming Platform - Cloudflare Worker

YOYO多用户、多频道安全流媒体Web播放平台的Cloudflare Worker业务逻辑层

## 功能特性

- 🔐 **安全认证**: 基于Cookie的会话管理，支持管理员和普通用户角色
- 📺 **流媒体管理**: 完整的频道CRUD操作，支持按需转码
- 🌐 **HLS代理**: 安全的视频流代理，支持缓存优化
- 📊 **系统监控**: 实时系统状态监控和健康检查
- 🚀 **高性能**: 基于Cloudflare网络的全球CDN加速
- 📝 **结构化日志**: 完整的操作日志和审计追踪
