## 🐛 常见问题排查

### 问题1：页面白屏
**可能原因：**
- 环境变量配置错误
- 依赖安装不完整
- 构建配置问题

**解决方案：**
1. 检查浏览器控制台错误信息
2. 确认环境变量配置
3. 重新安装依赖：`rm -rf node_modules && npm install`

### 问题2：API请求失败
**可能原因：**
- 后端服务未启动
- API代理配置错误
- CORS问题

**解决方案：**
1. 确认后端Cloudflare Worker服务正常
2. 检查`vite.config.js`中的代理配置
3. 验证环境变量中的API地址

### 问题3：视频播放失败
**可能原因：**
- HLS流地址无效
- 网络连接问题
- 浏览器兼容性

**解决方案：**
1. 检查HLS流地址是否可访问
2. 确认浏览器支持HLS播放
3. 查看播放器错误日志

### 问题4：登录失败
**可能原因：**
- 测试账号不存在
- 密码错误
- 会话管理问题

**解决方案：**
1. 确认使用正确的测试账号
2. 检查后端用户数据
3. 清除浏览器缓存和Cookie
